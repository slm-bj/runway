# Ideas Directory

这个目录用于存放待投票的创意提案。

This directory is for idea proposals awaiting votes.

## 如何提交创意 (How to Submit an Idea)

1. 复制 `TEMPLATE.md` 创建你的创意文件
2. 按照模板填写内容
3. 提交 Pull Request
4. 等待社区投票

1. Copy `TEMPLATE.md` to create your idea file
2. Fill in the content following the template
3. Submit a Pull Request
4. Wait for community voting

## 文件命名规范 (File Naming Convention)

- 使用小写字母和连字符
- 格式: `idea-name.md`
- 示例: `ai-code-reviewer.md`, `blockchain-voting-system.md`

- Use lowercase letters and hyphens
- Format: `idea-name.md`
- Examples: `ai-code-reviewer.md`, `blockchain-voting-system.md`
