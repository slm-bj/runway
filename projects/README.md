# Projects Directory

这个目录用于存放已实施的项目。

This directory is for implemented projects.

## 项目结构 (Project Structure)

每个项目应包含以下文件：
Each project should contain the following files:

```
project-name/
├── README.md          # 需求描述和项目概述 (Requirements and project overview)
├── proposal.md        # 调研报告 (Research report)
└── REPOSITORY.md      # 项目仓库链接 (Project repository link)
```

## 如何添加项目 (How to Add a Project)

1. 确保你的创意已获得足够投票
2. 创建项目目录，使用项目名称
3. 按照 `Example/` 的格式创建三个必需文件
4. 提交 Pull Request
5. 更新主 README.md 的项目列表

1. Ensure your idea has received enough votes
2. Create a project directory using the project name
3. Create the three required files following the `Example/` format
4. Submit a Pull Request
5. Update the project list in the main README.md

## 示例参考 (Example Reference)

请参考 `Example/` 目录了解项目文档的格式和内容。

Please refer to the `Example/` directory to understand the format and content of project documentation.
